Andromeda

Introducing Andromeda: Emotion Tracking Andromeda is a special project I started to better understand my own mental and emotional states. It’s a simple app that turns feelings into colourful 3D spheres. These spheres show up based on the location where you felt them. The size of each sphere tells you how recent the emotion was, and different colours represent different emotions, creating a visual diary of feelings over time. Key Features: 3D Emotion Map: Emotions are shown as spheres in a 3D space, making it easy and fun to see how your feelings change. Tracking Location and Time: The app records where and when you felt each emotion, adding more detail to your emotional diary. Built with Cool Tech: I used JavaScript and Three.js for the graphics that make the 3D map, and Flask, a Python program, to run everything on the server.
